# Onboarding bank

Boas vindas ao banco do biopark, biobank!

## Comandos

Para começar utilize os comandos para:

- Instalar: `npm install`
- Buildar: `npm run build`
- Rodar build: `npm run build` e depois `npm run start`

## Tecnologias

- Next 14
- Prisma JS
- Vercel (deploy)
